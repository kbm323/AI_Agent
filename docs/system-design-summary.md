## Product

AI_Agent is a Discord-based virtual AI company orchestration project centered on
OpenClaw and reviewed by Hermes.

The system is for AI-assisted content production:

- music video planning
- character visuals
- VFX and Unreal implementation ideas
- automation and Discord bot work
- marketing strategy
- risk, legal/IP, and brand review

## Core Philosophy

The system is not:

```text
question -> one AI -> answer
```

The target workflow is:

```text
question
  -> task analysis
  -> team routing
  -> expert persona execution
  -> Hermes review and objection
  -> convergence evaluation
  -> OpenClaw final synthesis
  -> user approval or execution
```
